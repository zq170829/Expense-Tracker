# Expense Tracker
This project is built based on MERN stack. 
## Features
- Register/Login/Logout
- Add/Delete/Update/Get Expense when user is logged in.


## Tech

- React
- Express
- MongoDB
- Node.js

## Files Navagation
- backend
    - config
        - db.js
    - controllers
        - expenseController.js
        - userController.js  
    - middleware
        - authMiddleware.js
        - errorMiddleware.js
    - models
        - ExpenseModel.js
        - userModel.js
    - routes
        - expenseRoutes.js
        - userRoutes.js
   - server.js
    
- frontend




## Installation

Install backend dependencies
```sh
npm install
```
Install client dependencies
```sh
npm install
```

Run app in development (frontend & backend)
```sh
npm run dev
```
